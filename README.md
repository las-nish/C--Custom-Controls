# C# **Custom** Controls Collection

Designed to be similar to controls in ***Visual Studio***\
**Dark** and **Light** modes are available

List of the Controls
 - Custom Fonts
 - Buttons
 - Combo Box
 - Context Menu Strip
 - Labels
 - List View
 - Menu Strip
 - Progress Bars
 - Scroll Bars
 - Flat Tab
 - Terminal
 - Text Box
 - Text Editor
 - Tool Strip

**Note :** Some of the items here is incomplete.
